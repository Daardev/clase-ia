# Examples - Apple UI Skill

## Prompt de ejemplo 1

Disena una landing page SaaS con estilo minimalista premium, jerarquia clara, espaciado amplio, grilla responsive y CTA principal destacado. Usa composicion mobile-first y microinteracciones sutiles para botones y cards.

## Prompt de ejemplo 2

Crea una interfaz de dashboard con fondo neutro, tarjetas suaves, bordes redondeados y tipografia limpia. Mantiene contraste correcto AA y componentes escalables para desktop y tablet.

## Prompt de ejemplo 3

Construye una pagina de producto con hero simple, imagen destacada, bloques de beneficios y seccion FAQ. Prioriza claridad visual, ritmo vertical consistente y animaciones discretas sin sobrecargar la UI.
